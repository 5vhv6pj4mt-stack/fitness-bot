## ЧАСТЬ 1. ТЕКУЩЕЕ СОСТОЯНИЕ

### Стек

| Компонент | Технология |
|-----------|-----------|
| Бот | aiogram 3.13, Python 3.12 |
| БД | SQLite (fitness.db) через aiosqlite |
| AI анализ | Groq API (llama-3.3-70b-versatile) |
| Планировщик | APScheduler (напоминания о питании) |
| Web API | FastAPI + uvicorn (порт 8001) |
| Сервис | systemd fitness-bot.service |

### Путь и запуск

```bash
cd /home/oracle/fitness-bot

# Запуск бота
nohup venv/bin/python bot.py >> bot.log 2>&1 &

# Перезапуск через systemd
sudo systemctl restart fitness-bot

# Логи
tail -f bot.log
```

### Флоу тренировки

```
Список дней
    ↓ тап на день
Карточка дня (список упражнений с планом)
    ↓ «Начать»
Ввод подхода (вес / повторы / RPE / отдых)
    ↓ «✅ Подтвердить»
Авто-старт таймера отдыха
    ↓ «▶️ Следующий подход»
...цикл...
    ↓ последний подход
Итог: тоннаж / RPE / время / AI-анализ / прогрессия
```

#### Структура упражнения в программе

| Поле | Тип | Пример |
|------|-----|--------|
| exercise | string | «Жим штанги наклонной» |
| muscle_group | string | «Грудь» |
| sets | int | 4 |
| reps_range | string | «6–8» |
| weight | float | 50.0 |
| rpe_range | string | «8.5–9» |
| rest | string | «2м30с» |
| order_num | int | 0 |

#### Окно ввода подхода

```
┌──────────────────────────────────────┐
│ 1. Жим штанги наклонной              │
│ RPE 8.5–9  ·  отдых 2:30             │
│                                      │
│  1. 50кг×6–8  [+] 50×6 R8.5         │
│ >2. 50кг×6–8  [>] ввести             │
│  3. 50кг×6–8  [ ] ...                │
│  4. 50кг×6–8  [ ] ...                │
│            Подход 2/4                │
├──────────────────────────────────────┤
│  [−2.5]   [⚖️  50 кг]   [+2.5]      │
│  [−1]     [🔁  6 повт]   [+1]        │
│  [7] [7.5] [✓8] [8.5] [9]           │
│  [−30]    [⏱  2:30]    [+30]        │
│  [✅  50кг × 6  RPE 8  ·  ⏱ 2:30]  │
│  [🔥 Разминка] [📖 Техника] [🏁]    │
└──────────────────────────────────────┘
```

### Схема базы данных

```sql
users              — профиль, вес, цель
programs           — программы тренировок
program_days       — дни программы (1–33)
program_exercises  — упражнения в дне
exercises          — справочник упражнений
workouts           — тренировки пользователя
workout_sets       — подходы
user_program       — прогресс по программе
weight_log         — история веса
body_measurements  — замеры тела (грудь/талия/бёдра/руки/ноги)
encyclopedia       — статьи
nutrition_items    — продукты + КБЖУ (ккал/белки/жиры/углеводы на 100г)
```
